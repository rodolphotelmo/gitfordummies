Projeto GIT

  1. Visão geral do GIT

    2. Ele é um sistema do tipo VCS (Version Control System), um Sistema de Controle de Versão.

    Nos auxilia de forma segura a evolução de arquivos texto, muito utilizado pelos desenvolvedores para trabalho em conjunto em um mesmo projeto.

  2. Comandos Básicos
    3. $ git init Para iniciar um repositório na pasta atual
    $ git status $ git config --local user.name "Seu nome aqui"
    $ git config --local user.email "seu@email.aqui"
  3. Comandos Básicos 2
    4. $ git remote add origin https://github.com/rodolphotelmo/gitfordummies.git Vincula projeto a pasta corrente
    $ git add index.html
    $ git pull origin main
    $ git pull
    $ git diff
    $ git add ARQUIVO
    $ git push origin main
  4. Trabalhando com repositórios de terceiros
    1. Clonar o repositório com o "fork"
    2. Criar um diretório local para o projeto que irá clonar
    3. inicializa repositório para trabalhar com o git: $ git init
    4. Vincular o repositório com o comando "remote"
    5. $ git remote add main [url.repositório] main é o local que direciona ao repositório
    6. Selecionar o Branch: git checkout main main
    7. Baixar o repositório: $ git pull
    8. Nesse ponto estamos pronto para trabalhar com o código desse projeto
  5. Modificando o projeto e enviando nossas alterações
    1. Editar o código
    2. Vrificar status do projeto $ git status
    6. $ git commit -m "Mensagem do Commit"
    $ git log --oneline $ git rebase -i HEAD~3
